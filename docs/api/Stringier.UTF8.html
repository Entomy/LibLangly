Class UTF8

Helper functions for working with UTF-8 data.

Inheritance
Object
UTF8
Namespace: Stringier
Assembly: Stringier.Encodings.dll
Syntax
public static class UTF8 : object
Remarks

Most of these helpers assume working with a UTF-8 stream, not a buffer. This is important because it enables us to more easily support stream decoding. Buffered data doesn't need special considerations, so these operations still work with buffers.

NextUnitRange(Byte, out Byte, out Byte)

Gets the valid range the next unit after unit can be within.

Declaration
public static void NextUnitRange(Byte unit, out Byte lower, out Byte upper)
Parameters
Type Name Description
Byte unit

The to check.
Byte lower

The lower bound of the next .
Byte upper

The upper bound of the next .
Remarks

This can't possibly know when we are at the end of a sequence, so with the exception of special cases, it always outputs the range for a UTF-8 "tail". You need to use this together with SequenceLength(Byte) to know how long the sequence actually is. Because of how this is meant to be used, the range 0x00..0x7F does not return itself. When you have that, SequenceLength(Byte) will return 1 and you have no business using this method, so it's optimized to not even handle that case.
