Interface IWrite<TElement, TError>

Indicates the type can be written to.

Namespace: Langly
Assembly: Langly.Core.dll
Syntax
public interface IWrite<TElement, TError>
Type Parameters
Name Description
TElement

The type of the element to write.

The type of the error object.

Properties

| Improve this Doc View Source

Writable

Can this be written to?

Declaration
bool Writable { get; }
Property Value
Type Description
Boolean
Remarks

This is a state indicator. Of course an IWrite<TElement, TError> can be written to, but that doesn't mean it can always be written to. Rather, this being true indicates the type can currently be written to.
