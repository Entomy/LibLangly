Writable

Can this be written to?

Declaration
bool Writable { get; }
Property Value
Type Description
Boolean
Remarks

This is a state indicator. Of course an IWrite<TElement, TResult> can be written to, but that doesn't mean it can always be written to. Rather, this being true indicates the type can currently be written to.
