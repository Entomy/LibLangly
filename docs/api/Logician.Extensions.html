Remarks

These, really, are just for binding purposes and orthogonality. Because all logic types are structs, and in fact required to be by the interfaces, there's no need for the null-tolerant extension pattern to be used. As such, Boolean is the only type that gains anything here. It does put everything in one consistent place for binding purposes, however.
