Remarks

This is intended as a reusable base to make implementing custom associative lists easier. It can not be used on its own, as it's abstract. Rather, you derive this type, and also derive a LinkedNode<TIndex, TElement, TSelf>, and pass your derived node into TNode. This will create most of your list for you, with all links being of TNode, not some base type that you have to upcast.
