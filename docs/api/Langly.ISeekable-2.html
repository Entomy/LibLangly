Seekable

Can this be seeked?

Declaration
bool Seekable { get; }
Property Value
Type Description
Boolean
Remarks

This is a state indicator. Of course an ISeekable<TElement, TError> can be seeked, but that doesn't mean it can always be seeked. Rather, this being true indicates the type can currently be seeked.
