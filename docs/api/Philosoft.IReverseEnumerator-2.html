Type Description

An enumerator that can be used to iterate through the collection in reverse.
Remarks

This exists so the C#/VB compilers duck typing can identify a call to GetReverseEnumerator() as itself returning an IEnumerable<T> that it can use. Without this, foreach won't work. It also degeneralizes the call to avoid a heap allocation.
