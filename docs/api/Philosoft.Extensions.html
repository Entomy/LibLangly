Parameters
Type Name Description
ICloneable<TSelf> collection

This collection.
Returns
Type Description
TSelf

A new object that is a copy of this instance.
Type Parameters
Name Description
TSelf

The type of this collection; itself.
Remarks

The resulting clone must be of the same type as, or compatible with, the original instance.

An implementation of Clone<TSelf>(ICloneable<TSelf>) can perform either a deep copy or a shallow copy. In a deep copy, all objects are duplicated; in a shallow copy, only the top-level objects are duplicated and the lower levels contain references. Because callers of Clone<TSelf>(ICloneable<TSelf>) cannot depend on the method performing a predictable cloning operation, we recommend that ICloneable<TSelf> not be implemented in public APIs.

See MemberwiseClone() for more information on cloning, deep versus shallow copies, and examples.
