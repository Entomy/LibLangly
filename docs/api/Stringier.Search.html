Performs a Rabin-Karp string-search, finding the starting index of the search if found in source.

Declaration
public static Int32 RabinKarp(String source, ReadOnlySpan<Char> search)
Parameters
Type Name Description
String source

The text to search in.
ReadOnlySpan<Char> search

The text to search for.
Returns
Type Description
Int32

The index at which search was found to start; -1 if not found.
Remarks

This searches for search through a rolling-hash, which keeps the search effective while not requiring precomputing tables.
