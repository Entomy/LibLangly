Func<TElement, TElement> func

The Func<T,TResult> to apply to each element.
Remarks

This is a mutating operation. It mutates the collection rather than constructing a new collection for the result. This is done for two reasons. First is that we can't generalize Map(Func<TElement, TElement>) like this while also allowing arbitrary collections to be constructed. Second is performance, as mutating in-place is substantially more efficient. Langly is not a pure functional programming library, it just takes inspiration in the form of more practical application of many functional principals.
