Type Name Description
Memory<Association<TIndex, TElement>> members

The members to enumerate over.
nint length

The length of the members.
Remarks

At first it might seem like length is superfluous, as Array has a known length. However, many data structures use an array as an allocated chunk of memory, with the actual array as a portion of this, up to the entire chunk. length is the actual used portion.
