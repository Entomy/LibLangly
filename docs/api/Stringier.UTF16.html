Class UTF16

Helper functions for working with UTF-16 data.

Syntax
public static class UTF16
Remarks

Most of these helpers assume working with a UTF-16 stream, not a buffer. This is important because it enables us to more easily support stream decoding. Buffered data doesn't need special considerations, so these operations still work with buffers.

NextUnitRange(Char, out Char, out Char)

Get the valid range the next unit after char can be within.

Declaration
public static void NextUnitRange(char char, out char lower, out char upper)
Parameters
Type Name Description
Char char

The Char to check.
Char lower

The lower bound of the next Char.
Char upper

The upper bound of the next Char.
Remarks

This is meant to be similar in design to NextUnitRange(Byte, out Byte, out Byte) and so it works the same way, even though it could be made not to for UTF-16 specifically. This variant gets a specific range for the next code unit when possible, and otherwise returns the full range from 0x0000..0xFFFF.
