Hamming(Char*, Int32, Char*, Int32, Level)

Calculates the Hamming edit-distance between the text and other.

Declaration
[CLSCompliant(false)]
public static int Hamming(char *text, int length, char *other, int otherLength, Level level)
Parameters
Type Name Description
Char* text

The first text.
Int32 length

The length of the text.
Char* other

The other text.
Int32 otherLength

The length of the other.
Level level

The Level of text elements to measure.
Returns
Type Description
Int32

The number of edits to get from text to other.
Remarks

Hamming edit-distance measures the amount of substitutions between two texts. Because it does not handle insertions or deletions, the two texts are required to be the same length. It also does not handle transpositions, so a transposition will be counted as two substitutions. This is the simplest of all EditDistance algorithms, and is fast, but limited.
