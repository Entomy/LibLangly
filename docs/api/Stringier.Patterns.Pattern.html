Consume(Char*, Int32, ref Int32, out Exception, IAdd<Capture>)

Calls the consume parser for this Pattern on the source.

Declaration
[CLSCompliant(false)]
protected abstract void Consume(char *source, int length, ref int location, out Exception exception, IAdd<Capture> trace)
Parameters
Type Name Description
Char* source

The source to parse.
Int32 length

The length of the source.
Int32 location

The location within the source to begin parsing, updated to the end of the match.
Exception exception

The exception that occurred during parsing, if any.
IAdd<Capture> trace

The object to trace the steps through the pattern graph in.
Remarks

This looks in source at the location for this Pattern, advancing location to the position just after the match, if found, and leaving it unchanged if not found. exception is null if a match is found, and assigned with information on the failure that occurred. If trace is a valid object, it is used to collection steps through the pattern graph for debugging purposes.

Neglect(Char*, Int32, ref Int32, out Exception, IAdd<Capture>)

Calls the neglect parser for this Pattern on the source.

Declaration
[CLSCompliant(false)]
protected abstract void Neglect(char *source, int length, ref int location, out Exception exception, IAdd<Capture> trace)
Parameters
Type Name Description
Char* source

The source to parse.
Int32 length

The length of the source.
Int32 location

The location within the source to begin parsing, updated to the end of the match.
Exception exception

The exception that occurred during parsing, if any.
IAdd<Capture> trace

The object to trace the steps through the pattern graph in.
Remarks

This behaves identically to Consume(ReadOnlyMemory<Char>, ref Int32, out Exception, IAdd<Capture>), except that a match is defined as anything the same length as this Pattern, but not equal to it. If the source is the same length and content as this Pattern, it is considered a failed match. This is used to implement Not().
