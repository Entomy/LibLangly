Indicates the type is a sequence of TMember.

public interface ISequence<TMember, out TEnumerator> : IContains<TMember>, ICount, IEnumerable<TMember>, IEnumerable where TEnumerator : IEnumerator<TMember>
Type Parameters
Name Description
TMember

The type of elements in the sequence.

The type of the enumerator for this sequence.
Remarks

This interface devirtualizes the enumerator, and simplifies numerous parts of the interface through well defined defaults.

The function describing the folding operation. This is a magma.
TMember identity

The identity value for func.
Returns
Type Description
TMember

A single element after folding the entire collection.
Remarks

func is a magma, so associativity like left-fold and right-fold are completely irrelevant.

identity is required as a start point for the fold. It needs to be the identity of the func to function properly. For example, the identity of addition is 0, and the identity of multiplication is 1. Without an appropriate identity, the results will be wrong.
