Writable

Can this be written to?

Declaration
bool Writable { get; }
Property Value
Type Description
Boolean
Remarks

This is a state indicator. Of course an IWritable<TElement, TError> can be written to, but that doesn't mean it can always be written to. Rather, this being true indicates the type can currently be written to.

Remarks

This assumes the write will never fail, which is an assumption that generally doesn't hold up. It is strongly recommended to use TryWrite(TElement) or TryWrite(TElement, out TError) instead.
