Class Collection

Provides various operations to help with developing collections.

Inheritance
Object
Collection
Namespace: System.Traits.Concepts
Assembly: Philosoft.Providers.dll
Syntax
public static class Collection : object
Remarks

These are primarily intended for the creation of new collections, with any of the contained operations being "reasonable defaults" for the implementation of the equivalent public API.
