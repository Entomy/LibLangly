Map<TElement>(IIndex<nint, TElement>, Func<TElement, TElement>)

Applies the func to each element of the collection.

Declaration
public static void Map<TElement>(this IIndex<nint, TElement> collection, Func<TElement, TElement> func)
Parameters
Type Name Description
IIndex<nint, TElement> collection

This collection.
Func<TElement, TElement> func

The Func<T,TResult> to apply to each element.
Type Parameters
Name Description
TElement
Remarks

This is a mutating operation. It mutates the collection rather than constructing a new collection for the result. This is done for two reasons. First is that we can't generalize Map<TElement>(IIndex<nint, TElement>, Func<TElement, TElement>) like this while also allowing arbitrary collections to be constructed. Second is performance, as mutating in-place is substantially more efficient. Philosoft is not a pure functional programming library, it just takes inspiration in the form of more practical application of many functional principals.
