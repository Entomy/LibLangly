Class Console

Represents the standard input, output, and error streams.

public static class Console
Remarks

This is intended as a replacement for Console. It bypasses much of the .NET runtime functionality, instead binding to the native console API's directly. It also operates in a uniquely different way, going for the direct API's rather than operating as a stream.

Remarks

This is a special devirtualizing overload. If the type being written is both a value type and implements IStringable<TSelf>, then the devirtualized ToString() will be used instead of ToString(). This saves a heap allocation through boxing just to call a method.
