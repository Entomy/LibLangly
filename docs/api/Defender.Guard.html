Default<TEnum>(TEnum, String)

Guards the langword_csharp_default clause of a langword_csharp_switch statement.

Declaration
public static Exception Default<TEnum>(TEnum value, string name)

    where TEnum : struct, Enum
Parameters
Type Name Description
TEnum value

The TEnum value.
String name

The name of the argument.
Returns
Type Description
Exception

The appropriate Exception to throw.
Type Parameters
Name Description
TEnum

An Enum type.
Remarks

If FlagsAttribute is present, this guard is uneccessary and will even report false negatives.

Unlike most of Guard, this returns an Exception in order to create a syntax that the language analyzer will understand. Even if this method always threw, the analyzer isn't likely to see that, so by returning an exception that will be immediately thrown, it can see that the langword_csharp_default doesn't fall-through.
