Remarks

This is intended as a reusable base to make implementing associative nodes for custom lists easier. It can not be used on its own, both because it's abstract but also a CRTP receiver and would recurse if you tried to use it. Rather, you derive from this type, passing your derived type into TSelf, and all the node links will be of TSelf, not some base type that you have to upcast.
