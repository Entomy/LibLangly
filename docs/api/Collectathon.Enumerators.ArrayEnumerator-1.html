Type Name Description
ReadOnlyMemory<TElement> memory

The memory to enumerate over.
IntPtr length

The length of the active memory.
Remarks

At first it might seem like length is superfluous, as ReadOnlyMemory<T> has a known length. However, many data structures use an array as an allocated chunk of memory, with the actual array as a portion of this, up to the entire chunk. length is the actually used portion.
