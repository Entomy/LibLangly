Syntax
public abstract class CharStreamBase : StreamBase, IControlled, IDisposable, IRead<byte>, ISeek, IWrite<byte>, IAdd<byte>, IRead<char>, IWrite<char>, IAdd<char>
Remarks

CharStream doesn't necessarily use this, since it's derived from Stream and has to use any StreamBase. This is, instead, support for a common hot-path by optimizing those cases. If the CharStream recognizes it's using a CharStreamBase, these optimized paths will be taken.
