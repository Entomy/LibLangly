Type Parameters
Name Description
TElement

The type of the elements being counted.
Remarks

This isn't intended for counting a single thing over and over. Rather, it's meant for counting collections of things all at once, and then doing things with those counts.

This isn't derived from DataStructure<TIndex, TElement, TSelf, TEnumerator>, and instead uses its own hardcoded behavior. As such, custom filtration isn't possible, with it instead doing its own combination of unique and sparse filtration as appropriate.
