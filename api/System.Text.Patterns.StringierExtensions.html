A new Pattern which matches the negation of this Pattern.
Remarks

Negation has to be constrained in some way to make it work in a sensible and understandable way. This has been done by saying the negated match has to be the same length as the pattern, but not the same content. If a match would normally fail, it instead succeeds, consuming the same amount of positions as if the normal pattern succeeded.
